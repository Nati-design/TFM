# TFM
Código usado en el TFM: "Comparación de técnicas exactas y heurísticas para la optimización de rutas de empresas de reparto de última milla".

1. Generate_dataset.ipynb contiene el conjunto total de nodos y genera diferentes instancias
2. dataset_dhl_i.pkl, con i de 1 a 10, contiene las diferentes instancias para ejecutar.
3. Open_dataset.ipynb abre los datos y realiza modificaciones sobre ellos, dando lugar a los archivos dataset_i_actualizado.pkl.
4. dataset_i_actualizado.pkl, con i de 1 a 10, contiene las instancias modificadas.
5. main_program.ipynb resuelve el problema de optimización EVRT de forma exacta.
6. nearest_neighbour.ipynb resuelve el problema de optimización del EVRT con el algoritmo del vecino más próximo .
7. Lin-Kernighan.ipynb resuelve el problema de optimización EVRT con el algoritmo k-opt.
8. Representacions.ipynb contiene el código para las representaciones incluidas en el TFM.
